# StringView

## Описание

Реализация невладеющей обёртки над C-style строками, аналогичной std::string_view.

StringView хранит только указатель на данные и длину, не управляя временем жизни памяти.

---

## Цель

Реализовать легковесную структуру для работы со строками без копирования данных.

Ключевые цели:
- устранение array decay при передаче строк
- доступ к данным за O(1)
- поддержка базовых операций без std::string

---

## Интерфейс

### Конструкторы

- StringView()
  - пустой view

- StringView(const char* str)
  - создание view от C-style строки (// NOLINT)

- StringView(const char* str, size_t size)
  - создание view с явной длиной

---

## Методы доступа

- operator[](size_t index) — доступ к символу
- Front() — первый символ
- Back() — последний символ
- Data() — указатель на данные
- Size() — размер
- Length() — аналог Size()
- Empty() — проверка на пустоту

---

## Модификация view

- RemovePrefix(size_t n)
  - сдвигает начало строки

- RemoveSuffix(size_t n)
  - уменьшает длину строки

- Swap(StringView& other)
  - обмен содержимым

---

## Подстроки

- Substr(size_t pos, size_t count = npos)
  - возвращает view на подстроку
  - выбрасывает StringViewOutOfRange, если pos >= Size()

---

## Сравнение

### operator==

- сравнение по длине и содержимому
- без копирования данных
- линейная сложность O(n)

---

## Исключения

`cpp
class StringViewOutOfRange : public std::exception;

Особенности

НЕ владеет памятью

не копирует данные

работает поверх существующего буфера

операции доступа O(1)

сравнение O(n)



---

Ограничения

исходная строка должна оставаться валидной

нельзя модифицировать данные

поведение при nullptr ограничено конструкторами



---

Пример использования

const char* text = "hello world";

StringView sv(text);

StringView part = sv.Substr(6, 5); // "world"

if (part == "world") {
    // true
}

sv.RemovePrefix(6); // "world"
