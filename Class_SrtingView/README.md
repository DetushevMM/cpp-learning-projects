# StringView - Легковесное представление строки

## Обзор

**StringView** — это класс для эффективной работы со строками только для чтения. Он предоставляет "окно" просмотра на существующую строку без владения данными и без их копирования.

## ✨ Ключевые особенности

- **Легковесность**: Занимает всего 16 байт (указатель + размер), копируется быстро
- **Только чтение**: Не изменяет исходные данные
- **Без аллокаций**: Не выделяет память, не копирует строки
- **Универсальность**: Работает с C-строками и строковыми литералами

## 🎯 Когда использовать

- **Параметры функций**: Вместо `const std::string&` для избежания копирований
- **Парсинг и токенизация**: Разделение строк без выделения памяти
- **Просмотр подстрок**: Получение части строки за O(1)
- **Сравнение строк**: Эффективные операции сравнения

## 🏗 Конструкторы

| Конструктор | Описание | Пример |
|------------|----------|--------|
| `StringView()` | Пустой StringView | `StringView sv;` |
| `StringView(const char* str)` | Из C-строки (неявное преобразование) | `StringView sv = "hello";` |
| `StringView(const char* str, size_t size)` | Из указателя и размера | `StringView sv(ptr, len);` |

## 📚 Методы класса

### Информация о строке
| Метод | Описание |
|-------|----------|
| `size_t Size() const` | Возвращает длину строки |
| `const char* Data() const` | Возвращает указатель на данные |

### Доступ к символам
| Метод | Описание |
|-------|----------|
| `const char& Get(size_t i) const` | Доступ к символу по индексу |

### Модификация представления
| Метод | Описание |
|-------|----------|
| `void RemovePrefix(size_t delta)` | Удаляет `delta` символов с начала |
| `void RemoveSuffix(size_t delta)` | Удаляет `delta` символов с конца |
| `StringView Substr(size_t pos, size_t size)` | Создает подстроку (без копирования!) |

## 🔍 Операторы сравнения

Поддерживаются все операции сравнения:
- `<`, `>`, `<=`, `>=`, `==`

Сравнение происходит лексикографически, как у обычных строк.

## 💡 Примеры использования

```cpp
#include "StringView.hpp"
#include <iostream>

int main() {
    // Создание StringView
    StringView sv1("Hello, World!");
    StringView sv2 = "Hello";  // неявное преобразование
    
    // Доступ к символам
    std::cout << "Первый символ: " << sv1.Get(0) << '\n';
    std::cout << "Длина строки: " << sv1.Size() << '\n';
    
    // Создание подстроки (без копирования!)
    StringView sub = sv1.Substr(7, 5);  // "World"
    std::cout << "Подстрока: " << sub << '\n';  // через operator<<
    
    // Удаление префикса/суффикса
    StringView sv3("   trimmed   ");
    sv3.RemovePrefix(3);    // убираем пробелы спереди
    sv3.RemoveSuffix(3);    // убираем пробелы сзади
    std::cout << "Результат: '" << sv3 << "'\n";  // "trimmed"
    
    // Сравнение строк
    StringView a = "apple";
    StringView b = "banana";
    std::cout << "a < b: " << (a < b) << '\n';
    
    return 0;
}
```

## 📤 Вывод в поток

Класс поддерживает вывод через `operator<<`:

```cpp
StringView sv("Пример строки");
std::cout << sv << '\n';  // Выведет: Пример строки
```

## 📝 Примечания

1. **Безопасность**: Убедитесь, что исходная строка "живет" дольше, чем StringView
2. **Неизменяемость**: StringView не может изменять исходную строку
3. **Производительность**: Все операции O(1) кроме сравнения (O(n))
