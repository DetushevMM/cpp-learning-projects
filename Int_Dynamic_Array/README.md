# IntDynamicArray

Динамический массив целых чисел на C++ с ручным управлением памятью. Учебный проект, демонстрирующий фундаментальные концепции языка C++.

## 📝 Описание

`IntDynamicArray` — это реализация динамического массива (аналог `std::vector<int>`), созданная в учебных целях для понимания:

- Как работают динамические структуры данных "под капотом"
- Принципов управления памятью в C++ (`new`/`delete`)
- Правила трёх (Rule of Three)
- Работы с указателями

Проект полностью написан на C++ с использованием только стандартной библиотеки (`iostream`) и демонстрирует ручное управление памятью без использования контейнеров STL.

## ✨ Возможности

- ✅ Создание массива заданного размера
- ✅ Добавление элементов в конец (с автоматическим расширением)
- ✅ Удаление последнего элемента
- ✅ Доступ по индексу с проверкой границ
- ✅ Вывод содержимого массива
- ✅ Глубокое копирование (конструктор копирования)
- ✅ Безопасное присваивание (оператор присваивания)
- ✅ Автоматическое освобождение памяти (деструктор)
- ✅ Автоматическое увеличение размера при переполнении

## 💡 Примеры

### Базовое использование
```cpp
#include <iostream>
#include "IntDynamicArray.h"

int main() {
    // Создаём массив с начальным размером 3
    IntDynamicArray arr(3);
    
    // Добавляем элементы
    arr.Push(10);
    arr.Push(20);
    arr.Push(30);
    arr.Push(40);  // Автоматическое расширение!
    
    // Выводим массив
    arr.Print();  // [10, 20, 30, 40]
    
    // Получаем элемент по индексу
    int value = arr.Get(2);  // 30
    
    // Удаляем последний элемент
    arr.Pop();
    arr.Print();  // [10, 20, 30]
    
    return 0;
}
```

### Демонстрация глубокого копирования
```cpp
IntDynamicArray arr1(2);
arr1.Push(5);
arr1.Push(10);

// Конструктор копирования - создаём независимую копию
IntDynamicArray arr2 = arr1;

// Оператор присваивания
IntDynamicArray arr3;
arr3 = arr1;

// Изменяем оригинал
arr1.Push(15);

// Копии не изменились!
arr1.Print();  // [5, 10, 15]
arr2.Print();  // [5, 10]
arr3.Print();  // [5, 10]
```

### Обработка ошибок
```cpp
IntDynamicArray arr(2);
arr.Push(1);
arr.Push(2);

// Попытка доступа за границы
int value = arr.Get(5);  // Выведет ошибку, вернёт -1

// Попытка удалить из пустого массива
IntDynamicArray empty;
empty.Pop();  // Выведет сообщение об ошибке
```

## 🔍 Детали реализации

### Структура класса
```cpp
class IntDynamicArray {
private:
    int* data_;      // Указатель на данные в куче
    int size_;       // Текущее количество элементов
    int capacity_;   // Выделенный размер памяти
    
    void Resize();   // Увеличение вместимости
    // ... остальные методы
};
```

### Управление памятью
- **Выделение памяти**: `data_ = new int[capacity_];`
- **Освобождение памяти**: `delete[] data_;`
- **Копирование**: создаётся новый блок памяти, элементы копируются поэлементно

### Правило трёх (Rule of Three)
Класс реализует все три специальных метода-члена:
1. **Деструктор** — освобождает память
2. **Конструктор копирования** — создаёт независимую копию
3. **Оператор присваивания** — безопасно копирует данные (с проверкой на самоприсваивание)

### Автоматическое расширение
При переполнении (`size_ >= capacity_`) вызывается метод `Resize()`:
- Вместимость увеличивается в 2 раза
- Создаётся новый блок памяти
- Старые данные копируются
- Старая память освобождается

### Проверка границ
Метод `Get(int index)` проверяет:
- `index < 0` — отрицательный индекс
- `index >= size_` — выход за правую границу
