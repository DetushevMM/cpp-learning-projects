# Класс `String` — динамическая строка (аналог `std::string`)

Учебная реализация динамической саморасширяющейся строки на C++: инкапсуляция буфера, явное разделение **размера** (`size`) и **вместимости** (`capacity`), корректная работа с памятью и эффективное добавление символов в конец.

## Задача (постановка)

Требуется класс, по смыслу близкий к `std::string`:

- хранение длины строки в поле класса;
- автоматическое расширение буфера при дописывании;
- освобождение памяти в деструкторе и при присваивании;
- **мультипликативная схема роста**: при переполнении буфера вместимость увеличивается **не менее чем в 2 раза** (в проверяющей системе ожидается множитель 2), чтобы суммарное число копирований при последовательных `PushBack` оставалось **линейным**, а не квадратичным.

Исходники по условию курса: `string.hpp`, `string.cpp`.

## Модель данных

- **`data_`** — указатель на динамический буфер; для пустой строки по условию может быть `nullptr`.
- **`size_`** — число значащих символов (без завершающего `'\0'`).
- **`capacity_`** — сколько символов помещается в выделенном буфере; всегда `capacity_ >= size_`.

В буфере после `size_` символов хранится нулевой байт `'\0'`, чтобы указатель из `CStr()` / `Data()` можно было использовать как C-строку (для непустой строки).

## Интерфейс (кратко)

### Конструкторы

- по умолчанию — пустая строка;
- `(size_t size, char symbol)` — строка заданной длины из повторяющегося символа;
- `(const char*)` — из C-строки, **не** `explicit` (по условию; в коде пометка `// NOLINT` для статического анализа);
- `(const char*, size_t)` — копирование не более `size` первых символов (с учётом возможного `'\0'` внутри отрезка).

### Правило пяти

Деструктор, копирующий/перемещающий конструктор и копирующее/перемещающее присваивание — владение динамическим массивом через `new[]` / `delete[]`.

### Доступ и свойства

`operator[]`, `Front` / `Back` (const и не const), `CStr`, `Data`, `Empty`, `Size`, `Length`, `Capacity`.

### Модификация

`Clear` (длина 0, **память не освобождается**), `Swap`, `PopBack`, `PushBack`, `operator+=` для другого `String`, `Resize`, `Reserve`, `ShrinkToFit`.

### Операторы

`operator+`, лексикографические `<`, `>`, `<=`, `>=`, `==`, `!=`, вывод в `std::ostream` и ввод из `std::istream`.

### Ввод из потока (`operator>>`)

- пропуск ведущих пробельных символов;
- чтение до **первого пробельного** символа после значимых;
- для проверки пробела используется `std::isspace` с **`static_cast<unsigned char>`**;
- символы читаются, например, через `std::istream::get` и добавляются через `PushBack`.
